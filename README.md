# React Native Todo App

A simple Todo application built using **React Native (CLI)** with **TypeScript** and **Redux**.

## Features

- Fetch todos
- Add new todo
- Edit todo
- Delete todo
- Mark as complete / incomplete
- Filter (All / Active / Done)
- Basic sorting
- Clean dark UI

## Tech Stack

- React Native CLI
- TypeScript
- Redux + Thunk
- React Navigation

## Project Structure
